Credit Risk Grade (CRR)  [TRM-176 Term]
     Open in Win Client    
Term Credit Risk Grade (CRR)
Description
Since the country in which the system will be deployed doesn’t use credit rating system, we have developed our own proprietory Credit Risk Grading modelwhich is driven by several parameters such as borrower’s socio-economic factors, financial status, etc. This grading system will be calculated and used to determine a minimum interest rate for a particular loan application.
  1. The system must have a Risk Grading setting page in the admin to set up this grading system.
  2. Each form field of the Borrower details will have an assigned Risk point (integer value) which will be used to calculate the Credit Risk Grade (A to F).
  3. The admin must be able to have access to the borrower details form fields in order to define the formula.
  4. E.g. assuming the formular for calculating Riks Grade is:
  5. 0.5*age + 0.7*marital status + 0.8*homeowner ; where age, maritalStatus, homeOwner
